Startr Js

StartrJs is a fast, small, and feature-rich JavaScript framework. It makes things like HTML document traversal and manipulation, event handling, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, StartrJs has changed the way that 1 of people write JavaScript.

A Brief Look

DOM Traversal and Manipulation

Get the <button> element with the class 'continue' and change its HTML to 'Next Step...'

setTxt("txt","A sample Code!");

Event Handling

Show the message element that is hidden with display:none in its CSS when <button> is clicked.

btnclk("sub",()=>{ show("msg") });

Ajax Request

Call a local script on the server /api/checkUser with the query parameter userId = uid_1011 and check user already exists or not, with the returned respons text.

Ajax({
    url:  "/api/checkUser",
    method: "POST",
    data: {
    st: true,
    userId: "uid_1011"
     },
    success: (respons) => {
     setTxt("status",respons)
   }
})


Docs

How to use StartrJs.  Full Documentation.


select(id)         // id - Any HTML tag id.

use :     select("t")       //   t   ->   <p id="t">Hello</p>.

getVal(id)         // id - Any HTML tag id.

use :     getVal("name")       //   name   ->   <input type="text" id="name"></input>.

setTxt(id,msg)       // id - Any HTML tag id.  | msg  ->  set your text here

use :     setTxt("span","Sample Text.")       //   span   ->   <span id="span">Sample Text.</span>.

setHtml(id,html)       // id - Any HTML tag id.  | html  ->  code html here

use :     setHtml("t1", '<b>Sample Text.</b>')       //   t1   ->   <p id="t1">Sample Text.</p>.

btnclk(id,event)       // id - Any HTML tag id.  | event  ->  code event function.

use :     btnclk("t",()=> {
   setTxt("span","Sample Text.")
})       //   t   ->   <button id="t"></button>.

addClass(id,class)       // id - Any HTML tag id.  | class  ->  set css class

use :     addClass("t","phone")       //   t   ->   <p id="t" class="phone"></p>.

removeClass(id,class)       // id - Any HTML tag id.  | class  ->  remove css class

use :     removeClass("t","phone")       //   t   ->   <p id="t"></p>.

show(id)         // id - Any HTML tag id.

use :     show("t")       //   t   ->   <p id="t" style="display:block;"></p>.

hide(id)         // id - Any HTML tag id.

use :     hide("t")       //   t   ->   <p id="t" style="display:none;"></p>.

txtcolor(id,color)       // id - Any HTML tag id.  | color  ->  color name

use :     txtcolor("t","red")       //   t   ->   <p id="t">Sample Text.</p>.

Ajax(url,method,data,respons)   // url - like.php, main.asp, dex.js  | method  ->POST / GET | data = { st: 1, name: "admin" }

use :
    POST   :-
Ajax({
    url:  "/api/checkUser",
    method: "POST",
    data: {
    st: true,
    userId: "uid_1011"
     },
    success: (respons) => {
     setTxt("status",respons)
   }
})
   GET :- Ajax({
    url:  "/api/checkUser",
    method: "GET",
    success: (respons) => {
     setTxt("status",respons)
   }
})
